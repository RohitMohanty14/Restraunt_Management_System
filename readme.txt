Restaurant Management System
A full-stack web application built to manage restaurant operations like online ordering, table reservations, and administration.

## Customer Features :
Register and log in securely.
Browse the restaurant menu.
Add items to a shopping cart, update quantities, and remove items.
Checkout and pay for all items at once.
Reserve a table.

## Admin Features :
Log in to a separate, secure admin dashboard.
Manage all users, menu items, orders, and reservations.
Full control to create, view, update, and delete menu items.

Technology :
Backend: Spring Boot MVC, Spring Security (with JWT)
Frontend: Thymeleaf, HTML, CSS
Database: JPA with MySQL
Testing: JUnit 5, Mockito
